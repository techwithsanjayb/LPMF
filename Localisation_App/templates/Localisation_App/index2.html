gDoc Plug-in
Overview
gDoc Add-in for Ms Word is a plug-in which can translate English data in word documents to Indian language with just one click using gistlangserver.in. The Translations provided by this extension are domain specific.User can also login and contribute to the data they want to in the Indian languages. While translation, it retains various formatting of the document like Bullets, font attributes, image, table..etc.It is currently available for Microsoft word 2007 and above.
Setting in Microsoft Word Document for enabling AddIn
1) Click the Microsoft Office Button Office button , and then click Word Options. 2) Click Trust Center, click Trust Center Settings, and then click Macro Settings. 3) Click the options:- Enable all macros (not recommended, potentially dangerous code can run)
Also check option:- Trust access to the VBA project object model
Application Support and Prerequisites for installing gDoc Add-Ins
To use this plug-in, internet connectivity is must for the plug-in to function properly. Office versions :- 2007, 2010, 2013, 2016 Office architecture :- 32-bit and 64-bit versions of Office Windows versions :- 7(SP1), 8, 8.1, 10 Windows architecture :- 32-bit and 64-bit Browser support :- Microsoft Internet Explorer 10+
